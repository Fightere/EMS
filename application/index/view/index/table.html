中国航发航材院研究员曹春晓称，将来C919可能要用的长江-1000发动机正在研制之中。

　　

曹春晓同时还是国家大型飞机重大专项专家咨询委员会委员、材料科学家、钛合金专家。

　　

曹春晓透露，长江-1000发动机的钛合金用量初步估算是23%。

　　

据了解，钛合金密度比钢小得多，强度和钢很接近，可以大大减轻飞机及其发动机的重量。

　　

据中国航空发动机集团官网，1964年，曹春晓与同事共同提出以TC4钛合金叶片和盘研制作为我国航空用钛的突破口。1965年，他率领课题组制成某发动机第一级压气机转子叶片。

　　

曹春晓称，钛合金是“航空材料大家族”中的“重要成员”，而发动机在航空装备中起着重要作用。“钛在上世纪50年代问世，大大促进了航空发动机的发展。航空发动机（对钛）的需求，强有力地推动了钛的发展。”

　　

在曹春晓看来，航空发动机是高科技的结晶，是创新型国家的重要标志，也是国家综合国力的集中表现。

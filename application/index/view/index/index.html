预约指南

所有在校老师可以通过本软件进行实验室的预约，并生成任务书，管理员可查看预约列表，以及修改任务书的送达状态、审核状态，生成课表。管理要可以增删改实验室属性（最大学生人数，各个课程支持的机器数等）以及执行实验室规章制度，增删改实验室值班表。

  • 1.点击“实验室预约”，可根据周数、实验地点选择查看实验室预约表格。（选择实验室以后，弹出该实验室规章制度）
  • 2.表格分为可预约（绿色）和不可预约（红色），鼠标所到单元处显示：实验课程名称、专业班级、人数、指导教师、可用实验设备数、总实验设备数。
  • 3.若单元格可预约，则可选择预约时间：上午：前两课时、后两课时；下午：前两课时、后两课时；晚上：前两课时、后两课时；（晚上第二学时不单独预约，必须同第一学时一起预约），输入预约信息：学期、课程名称和课号、所属教研室、实验学时、实验人数、每组人数，循环次数、实验类型、专业及班级、备注。
